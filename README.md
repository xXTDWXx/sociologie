# Bouwstenen van sociologie - oefenapp

Open `index.html` in je browser om te oefenen.

Wat zit erin:

- vragen per hoofdstuk
- meerkeuzevragen met feedback
- open examenvragen met modelantwoord
- casussen met rubric voor zelfcorrectie
- begrippentraining
- gemengde examenmodus
- lokale voortgang via de browser

De app werkt volledig lokaal en gebruikt enkel de leerstof die uit de cursus werd verwerkt.
